step 1. install termcolor from pypi

pip install -r requirements.txt
or...

step 2. install termcolor from gbruteforcer main folder

cd termcolor
sudo python setup.py install

step 3. change to gbruteforcer main directory

cd ../

step 4. run the script

python bruteforcer.py

NB>> you must type in your target email in single or double qoute
i.e "example@email.com", 'example@email.com'
same thing goes with the path to password list
i.e "path/to/password/list.txt", 'path/to/password/list.txt'
